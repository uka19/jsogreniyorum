JavaScript Değişken Oluşturma

Bir programlama dilinin temellerini değişkenler oluşturur. Değişkenler verilerin saklandığı birimlerdir.

Örnek

<script> var deneme = "Uka"; </script>

Değişkenler veri türü ve değer olmak üzere iki temel unsurdan oluşur. Örneğin; yukarıdaki örneğimizde değişken adımız deneme'dir. Veri türümüz string'dir. Yani metinseldir. Değerimiz ise Uka'dır.

Değişken ismi tanımlanırken bazı kurallar vardır:

Örnek

<script> var yağmur = null; // Değişken isimlerinde Türkçe karakter kullanılmaz. var Deneme = null; // Büyük - küçük harf ayrımı vardır. Deneme ile deneme aynı değişken değildir. var 300spartali = null; // Değişken isimleri nümerik değerler ile başlayamaz. </script>

Değişken adını vermeden önce tanımladığımız var ifadesi variable'dan gelmektedir ve dilimizde değişken anlamındadır.
