<Div> Alanı

Elementi çıkarmak için üzerine bas

Örneğimizde, yazdığımız yazının dinamik olarak bir div içerisine <p> elementi olarak eklendiğini görüyoruz.
Peki adım adım bunu nasıl başardık?

İlk olarak div bloğumuzu oluşturalım.

Örnek

<script> <div id="newElementZone" align="center"> <h2><Div> Alanı</h2> </div> </script>

burada id olarak newElementZone adını verdik. Bu id bizim o elemente erişebilmemiz için önemli.

İkinci olarak javaScript kısmımıza geçelim ve kodlarımızın ne işe yaradığını açıklayalım.

Örnek

<script> var metin = document.getElementById("textForNewElement").value; //Metnin içerisindeki veriyi metin değişkenine atadık. var text = document.createTextNode(metin); //Texti element olarak eklemeden önce Node oluşturmak gerekli. var newElement = document.createElement("P"); // Bir <p> elementi oluşturduk ve newElement değişkenine atadık. newElement.id = "newP" // Oluşturduğumuz elemente id atayabiliyoruz. Bu id o elemente erişmemiz için lazım olabilir. newElement.appendChild(text); // Oluşturduğumuz Node'yi yeni elementimize atadık. var contentOpt = document.getElementById("newElementZone"); //Başta oluşturduğumuz div elementini değişkene atadık. contentOpt.appendChild(newElement); // Sonunda eriştiğimiz div elementine p elementini eklemiş olduk. </script>

Adım adım temel olarak eklemeyi böyle yapabiliyoruz.

Peki gördüğümüz gibi eklediğimiz p elementlerini nasıl kaldırabiliriz ?
Bastığımız p elementini kaldırmak için bu elementi oluştururken onClick() fonskiyonunu bu elemente atamamız gereklidir.

Örnek

<script> newElement.onclick = function(){deletePFromId(newElement)}; </script>

Önceki örnekte p elementini oluşturmuştuk ve bunu newElement değişkenine atmıştık. Ondan sonraki satırda ise bu elemente onClick fonksiyonunu atadık ve parametre olarak o elementi yolladık ki bastığımızda, basılan element fonksiyonda işleme girsin

Şimdi örnek olarak bizim sitemizde bulunan kod paylaşma bloğunu nasıl JavaScript ile ekleyebiliriz onu gösterelim

<Div> Alanı

Kaldırmak için kod bloğunun üzerine bas

İlk olarak bir div alanı oluşturalım.

Örnek

<script> <div id="newElementZone2" align="center"> <h2><Div> Alanı</h2> </div> </script>

Şimdi ise javascript kodumuzu inceleyelim.

Örnek

<script> var elementZone = document.getElementById("newElementZone2"); //Oluşturduğumuz div alanını değişkenimize attık. var div = document.createElement("div"); //Kod bloğumuzun en dış div alanını oluşturduk. div.id = "codeblock"; //Oluşturduğumuz div alanına id'mizi atadık. Bu id css de önceden tasarlanmış idi. var h4 = document.createElement("h4"); //Başlığımız için h4 elementi oluşturduk. h4.innerHTML = "Örnek"; //h4 elementinin içine yazıyı yazdık. var insideDiv = document.createElement("div"); //Kod bloğumuzun içeri kısmındaki div alanını oluşturduk. insideDiv.id = "codeArea"; //Oluşturduğumuz son div'e id atadık. insideDiv.innerHTML = "Uka Code"; //Oluşturduğumuz son div içerisine yazı yazdık. div.appendChild(h4); //Sıra ile Kod bloğunun en dışındaki div'e başlığımız olan h4'ü atadık. div.appendChild(insideDiv); //İkinci sırada içerde oluşturmamız gereken div'i atmak var. elementZone.appendChild(div); // Son olarak en dıştaki div'i en başta oluşturduğumuz div içerisine attığımızda kod bloğumuz oluşmuş olmaktadır. </script>

Peki bu kod bloklarını nasıl kaldırabiliriz?

Örnek

<script> div.onclick = function(){deletePFromId2(div)}; </script>

Kod bloğunun en dışındaki oluşturduğumuz div için bu onclick fonksiyonunu atar isek, kod bloğunun üzerine her bastığımızda oluşturduğumuz bütün kod bloğu div alanından kalkacaktır

deletePFromId2() fonksiyonu da aşağıdaki gibi olması lazım

Örnek

<script> function deletePFromId2(p){ var element = document.getElementById("newElementZone2"); element.removeChild(p); } </script>
