Javascript Giriş

Javascript, web programlama ile ilgilenenler ve internet sitesi hazırlamaya başlamak isteyenler için bilmeleri gereken bir dildir. Javascript sayfalarınızın işlevsel olmasını sağlar. Kullanıcı ile tarayıcı arasında bir köprü görevi görür.

Örneğin bir html formda boş bırakılan yerlerin boş bırakılmaması gerektiğini javascript yoluyla kullanıcıya bildirebilirsiniz. Ya da sitenize tarih veya saat ekleyebilirsiniz. Javascript ile yapabilecekleriniz oldukça geniş. İşte bu sayfalarda neler yapabileceğinizi göreceksiniz.

Programlama Türkiye sayfalarının sonuncusu olan Javascript Dersleri ile temelinizi oluşturmaya hazırsınız. Keyifli kodlamalar!
